# Redux toolkit contact card

redux-toolkit-contact-card is a simple contact card crud app.
data is not persisted, but managed by State. 
Wire it to a Db to have persistance

# Introduction

A simple contacts card app build with React, Typescript, redux-toolkit 
Contacts can be Employee, or Company
Employees and Comanies can be part of an other company

# Installation Instructions

Clone the repository, change directories, and use NPM to install the dependencies.

```bash
$ git clone https://github.com/RaselJam/SaalContacts.git
$ cd redux-toolkit-contact-card
$ npm install
```

## Usage

- start development server with `npm start` or `sudo npm start`

* `npm start`

now The project can be viewed in the browser at

- [http://localhost:3000](http://localhost:3000)

The page will reload if you make edits.\
You will also see any lint errors in the console.
